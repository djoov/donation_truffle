Kebijakan Privasi

Transparansi adalah fondasi kepercayaan di DonasiKuy.

Terakhir Diperbarui: 24 November 2025
PENTING: Ini adalah Proyek Simulasi

Platform DonasiKuy dikembangkan untuk tujuan Edukasi & Penelitian Akademik. Semua transaksi menggunakan jaringan Ganache (Testnet Lokal) dan mata uang Ether (ETH) palsu. Jangan pernah menggunakan Private Key dari dompet Ethereum asli (Mainnet) Anda di sini.

1. Data yang Kami Kumpulkan

Untuk menjalankan layanan donasi terdesentralisasi ini, kami mengumpulkan beberapa informasi:

  • Informasi Akun: Username, Email, dan Password (terenkripsi).
  • Data Blockchain: Wallet Address (Public Key) dan Private Key (Untuk fitur Auto-Sign).
  • Data Kampanye: Judul, Deskripsi, Target Dana, dan Gambar pendukung.

2. Penggunaan Private Key

Kami menerapkan sistem Custodial Wallet untuk mempermudah pengalaman pengguna (UX). Artinya:

Bagaimana kami menggunakannya?

Private Key Anda disimpan di database lokal server hanya untuk tujuan menandatangani transaksi (Donasi/Buat Kampanye) secara otomatis tanpa perlu pop-up dompet eksternal.

3. Transparansi Blockchain

Harap diingat bahwa sifat dasar teknologi Blockchain adalah Transparan & Permanen (Immutable).

  • Semua transaksi donasi yang berhasil akan tercatat di buku besar publik (Ledger).
  • Siapapun dapat melihat jumlah donasi dan alamat pengirim (Wallet Address), namun identitas asli Anda (Nama/Email) tetap tersimpan aman di database kami (Off-chain).
  • Data yang sudah masuk ke Blockchain tidak dapat dihapus atau diubah oleh siapapun, termasuk Admin.

4. Keamanan Data

Kami menggunakan standar keamanan industri untuk melindungi data Anda:

Enkripsi SSL
Koneksi aman HTTPS.
Database Terisolasi
Akses terbatas hanya Admin.
